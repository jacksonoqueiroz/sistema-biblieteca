Sistema de Biblioteca, aluguel de livros.
-Cadastro de Leitores;
-Cadastro de Livros;
-Listagem de Livros;
-Listagem de Leitores;
-Listagem com botões de edição e exclusão;
-Livros com fotos;
-Aluguel com data de entrega de 15 dias;
-Envia notiicações por email (gmail), 
com aviso de entrega para o dia seguinte, e aviso de atraso,
informando os dias de atraso.
-Etiquetas de Leitores;
-Etiquetas de Livros com CDD padrão.
